# 🍽️ Week 08: Recipe Explorer

## 📝 What the project is about
In this project, you will build a **Recipe Explorer** application using React. This app allows users to search for their favorite recipes and explore cooking instructions, ingredients, and images fetched from a real-world API.

## 🎯 What you will learn
By completing this project, you will practice and master:
- **React Hooks:** Using `useState` for search queries and `useEffect` for API calls.
- **React Router:** Navigating between search results and detailed recipe instructions using dynamic routes.
- **Context API:** Managing a global state for user favorites or dietary preferences.
- **API Integration:** Fetching and handling data from an external REST API.
- **Dynamic UI:** Mapping through data to render recipe cards and handling loading/error states.
- **Component Architecture:** Building reusable UI components like search bars, lists, and loading spinners.

---

## 🚀 How to run it

### 1. Install Dependencies
Navigate to the project folder and run:
```bash
npm install
```

### 2. Start the App
Run the following command to start the development server:
```bash
npm run dev
```

Open the URL shown in your terminal to see your Recipe Explorer in action!
