# 🤖 Week 05: AI Prompts Library

## 📝 What the project is about
You will build a searchable library of high-quality AI prompts. This application helps users find the best instructions for AI models by category or search term, providing a curated database of prompts.

## 🎯 What you will learn
By completing this project, you will practice and master:
- **List Rendering:** Rendering large datasets efficiently using `.map()`.
- **Search & Filtering Logic:** Implementing real-time search functionality based on user input.
- **Props & State:** Passing data and update functions between parent and child components.
- **Component Styling:** Creating a clean, card-based UI for a library of data.

---

## 🚀 How to run it

### 1. Install Dependencies
Navigate to the project folder and run:
```bash
npm install
```

### 2. Start the App
Run the following command to start the development server:
```bash
npm run dev
```

Open the URL shown in your terminal to start exploring the AI Prompts Library!
